From Agentic AI Development to Systems Research

Abstract

Agentic AI development is broken. Too many tools, too many languages, too much glue code. Jaseci fixes this: an open-source language and runtime extending Python with native AI and scaling constructs. Prompt less, smile more, no need for DevOps.

In this tutorial, you will learn:

  • a How to build multi-agent AI systems using Jaseci's graph-based walker paradigm with zero boilerplate
  • b How the by llm() construct lets the compiler write prompts for you
  • c How Jaseci auto-deploys from laptop to Kubernetes with zero DevOps configuration
  • d How Jaseci's Python-transpiling frontend fixes PyTorch 2 FX graph breaks for ML compiler research
